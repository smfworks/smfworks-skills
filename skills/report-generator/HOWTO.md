# Report Generator — Quick Reference

## Install
```bash
smfw install report-generator
```

## Commands
```bash
smf run report-generator create                     # Interactive report wizard
smf run report-generator create --sample sales    # Create sample report
smf run report-generator templates                  # List templates
```

## Common Examples
```bash
# Create report with sample data
smf run report-generator create --sample sales

# List available templates
smf run report-generator templates

# Interactive report creation
smf run report-generator create
```

## Help
```bash
smf run report-generator help
```
