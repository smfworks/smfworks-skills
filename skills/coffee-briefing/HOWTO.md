# Coffee Briefing — Quick Reference

## Install
```bash
smfw install coffee-briefing
```

## Commands
```bash
python main.py briefing [date]           # Get your morning briefing
python main.py config                    # Configure API keys & preferences
python main.py add "Task item"          # Add item to briefing
python main.py list                      # List all briefing items
python main.py weather                   # Get weather only
python main.py news                      # Get news headlines only
```

## Common Examples
```bash
# Get your morning briefing
python main.py briefing

# Get briefing for a specific date
python main.py briefing 2026-03-25

# Add a task to your briefing
python main.py add "Review Q1 sales report"

# Check weather only
python main.py weather
```

## Help
```bash
python main.py --help
python main.py briefing --help
```
