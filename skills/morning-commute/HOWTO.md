# Morning Commute — Quick Reference

## Install
```bash
smfw install morning-commute
```

## Commands
```bash
python main.py commute                           # Get commute briefing
python main.py commute --home "New York"         # Set home location
python main.py commute --work "Boston"          # Set work location
python main.py traffic                          # Check traffic only
python main.py weather                          # Weather for commute
```

## Common Examples
```bash
# Get your morning commute briefing
python main.py commute

# Set home and work for personalized routes
python main.py commute --home "New York" --work "Boston"
```

## Help
```bash
python main.py --help
python main.py commute --help
```
